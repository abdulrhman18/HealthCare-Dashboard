## 📖 Overview
This repository contains a comprehensive, interactive Power BI dashboard designed to analyze healthcare facility data. It breaks down complex hospital metrics into actionable insights across patient demographics, clinical operations, and financial performance.

## ✨ Key Features
The dashboard is divided into three main actionable views:
1. **Overview:** Tracks patient demographics, total visits, admission trends, and leading diagnoses.
2. **Clinical:** Monitors operational efficiency, geographic patient distribution, and provider performance alongside patient satisfaction scores.
3. **Financial:** Analyzes revenue streams, treatment costs, and financial health categorized by diagnosis and geographic region.

## 💡 Key Insights Discovered
* **Patient Volume:** The facility handled nearly 5K patients, with a significant surge in visits during January.
* **Top Diagnosis:** Hypertension is the leading diagnosis and the most lucrative, generating $1.33M in total revenue.
* **Regional Concentration:** Wales drives the highest volume of visits (1.9K) and the most revenue ($1.30M).
* **Quality of Care:** Analysis revealed an inverse relationship between a doctor's patient load and average patient satisfaction scores, highlighting an opportunity to improve care through better caseload distribution.

## 🛠️ Tools & Technologies Used
* **Power BI Desktop:** Data visualization, interactive dashboard creation, and data modeling.
* **Power Query:** Data cleaning, shaping, and transformation.
* **DAX (Data Analysis Expressions):** Creating custom measures, calculated columns, and advanced aggregations.

## 🚀 How to Use This Dashboard
1. Download the `Healthcare_Dashboard.pbix` file from this repository.
2. Open the file using **Power BI Desktop** (available for free from Microsoft).
3. Explore the different tabs to interact with the visuals, apply filters, and slice the data.
